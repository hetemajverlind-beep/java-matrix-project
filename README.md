# java-matrix-project
A Java project that takes a user-defined square matrix as input and calculates its determinant using 2D arrays, nested for-loops, and well-structured methods, showcasing fundamental programming concepts like array manipulation, modular design, and algorithmic problem-solving.
Project : MATRIX -> Determinat .
Tech used : JAVA -> Methods -> Arrays(1D,2D) -> Nested Loops(for) .
